# Creaciones fantásticas y crónicas urbanas

Blog estático listo para subir a GitHub y activar con GitHub Pages.

## Qué contiene

- `index.html`: página principal del blog.
- `posts/`: entradas individuales del blog.
- `css/styles.css`: estilos visuales.
- `assets/img/`: imágenes usadas en las entradas.

## Qué se publicó

Se publicaron solo productos finales: cuentos, textos creativos, personaje e imágenes correspondientes. No se incluyeron instrucciones del taller, preguntas guía, ejemplos de clase ni material de referencia que no era creación final.

## Cómo subirlo a GitHub

1. Crear un repositorio nuevo en GitHub.
2. Subir todos los archivos y carpetas de este proyecto.
3. Ir a Settings > Pages.
4. En Branch seleccionar `main` y carpeta `/root`.
5. Guardar y esperar el enlace publicado.

El sitio funciona abriendo directamente `index.html`.
